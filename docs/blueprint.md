# **App Name**: Guardian Angel

## Core Features:

- SOS Signal: Allows the user to send an SOS signal with their current location to designated contacts.
- Chatbot Assistance: A chatbot provides immediate assistance and guidance during emergency situations.
- Live Location Sharing: Shares the user's live location with emergency contacts until help arrives.
- "Help is on the Way" Notification: Displays a reassuring message once the SOS has been triggered.
- User Feedback Collection: Collects user feedback on the app's performance and effectiveness.
- Hotel Suggestions: AI-powered tool that provides hotel suggestions based on Google review data; the tool determines when and if it is appropriate to suggest a hotel based on location and safety concerns.
- Nearby Resources Directory: Identifies and displays the locations of nearby hospitals and police stations.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to convey trust and safety.
- Background color: Light gray (#F0F4F7) for a clean, neutral base.
- Accent color: Soft orange (#FFA726) for key actions and alerts.
- Body and headline font: 'PT Sans', a humanist sans-serif that balances modernity with approachability, is appropriate for all on-screen text.
- Use clear, recognizable icons for SOS, location, and nearby services.
- Prioritize a clean, intuitive layout for ease of use in stressful situations.
- Subtle animations to guide the user and confirm actions.
